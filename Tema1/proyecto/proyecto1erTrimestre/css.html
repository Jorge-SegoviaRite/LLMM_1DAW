¿Qué es SVG?

  • SVG significa Gráficos Vectoriales Escalables (Scalable Vector Graphics)
  • SVG es un formato para describir imágenes vectoriales en XML
  • Permite crear gráficos que se pueden escalar sin perder calidad
  • Se puede usar para ilustraciones, iconos, gráficos y animaciones en la web
  • Los gráficos SVG pueden ser estilizados con CSS y manipulados con JavaScript

Un ejemplo básico de SVG

Ejemplo:

<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" />
</svg>

Explicación del ejemplo:

  • <svg> es el contenedor del gráfico SVG
  • width y height definen el tamaño del lienzo
  • <circle> crea un círculo
  • cx y cy son las coordenadas del centro del círculo
  • r es el radio del círculo
  • stroke define el color del borde
  • stroke-width define el grosor del borde
  • fill define el color de relleno

Ejemplos de atributos SVG

Atributo Descripción Ejemplo
fill Color de relleno de la figura fill="red"
stroke Color del borde stroke="blue"
stroke-width Grosor del borde stroke-width="3"
